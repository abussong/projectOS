# 🧠 RemindMind

**RemindMind** — это приложение для управления напоминаниями и задачами с поддержкой подзадач, приоритетов и кастомизируемых уведомлений.

## ✨ Возможности

### 🎯 Основной функционал
- Создание, редактирование и удаление задач
- Установка даты и времени выполнения
- Отметка о выполнении (чекбокс)
- Приоритеты задач: Высокий, Средний, Низкий
- Автоматическая сортировка по приоритету и дате

### 📋 Подзадачи
- Создание подзадач для любой основной задачи
- У каждой подзадачи своя дата, время и приоритет
- Отдельные чекбоксы для каждой подзадачи

### 🔔 Умные уведомления
- **Периодичность:** один раз, ежедневно, еженедельно
- **Выбор дней** для еженедельного повтора
- **Свой звук** для каждого напоминания
- **Вибрация** (вкл/выкл)
- **Повторы** уведомления (до 10 раз)

### 🎨 Интерфейс
- **3 темы оформления:** Светлая, Тёмная, Темно-синяя
- Автосохранение выбранной темы

### 🌍 Локализация
- Русский и английский язык (автоматически подстраивается под систему)
