## [2.0.0] - 2024-04-04

### Добавлено (Главное обновление)
- **Кастомизация уведомлений:**
  - Выбор периодичности (один раз, ежедневно, еженедельно)
  - Выбор дней недели для еженедельного повтора
  - Выбор звука уведомления из системных
  - Включение и выключение вибрации
  - Настройка количества повторов и интервала между ними
- Диалог настроек уведомлений
- Воспроизведение пользовательских звуков через MediaPlayer
- Сохранение настроек уведомлений в базе данных

### Исправлено
- Проблема с изменением звука уведомлений (решено через MediaPlayer)

## [1.0.0] - 2024-02-18

### Добавлено (Первый релиз)

## Схема версионирования

`X.Y.Z`

- **X** — крупные изменения (несовместимые с предыдущими версиями)
- **Y** — новый функционал (обратно совместимый)
- **Z** — минорные исправления (багфиксы)
